# Home Network & Homelab

## Overview
This repository documents my personal home network and homelab.
The goal is to design a secure, scalable, and well-documented edge network
using enterprise-style tooling.

## Goals
- Learn firewalling, VLANs, and routing
- Operate a fiber internet connection without ISP router
- Separate trusted, guest, and IoT networks
- Document decisions and trade-offs

## Architecture
- ISP: Deutsche Telekom FTTH (1 Gbit/s)
- Firewall/Router: Edge4Go (OPNsense)
- Switch: PoE managed switch
- Wi-Fi: UniFi access point

## Key Topics
- PPPoE + VLAN WAN
- Stateful firewalling
- VLAN-based segmentation
- Wi-Fi roaming
- PoE-powered infrastructure
